Чему научитесь

  1. Анализировать данные Освоите анализ данных с помощью сводных таблиц и фильтров
  2. Работать с функциями Узнаете все о встроенных функциях и инструментах форматирования
  3. Использовать горячие клавиши Изучите основные горячие клавиши и ускорите работу с помощью них
  4. Автоматизировать задачи Научитесь автоматизировать ваши рутинные задачи с помощью программы
